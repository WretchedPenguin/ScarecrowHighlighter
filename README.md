Scarecrow Highlighter - 1.0
Show the radius for the scarecrows effectiveness

:: REQUIREMENTS ::
• SMAPI v2.11

:: FEATURES ::
• Highlight on hovering over scarecrow
• Highlight on holding scarecrow
• Highlight on button toggle
